WorkFocus

웹캠 기반 집중도 분석, 회의 기록 자동화, RAG 검색을 한 화면에서 연결하는 AI 업무 집중 관리 대시보드입니다.

핵심 기능 보기 원본 프론트엔드

집중, 기록, 검색을 한 제품 흐름으로

기존 WorkFocus 프론트엔드와 백엔드 구조를 바탕으로, 실제 서비스 소개 페이지처럼 보이도록 정리한 정적 HTML 버전입니다.

F

집중 모니터링

웹캠 프레임을 분석해 집중, 자리 이탈, 스마트폰 사용, 졸음 이벤트를 기록합니다.

D

대시보드

오늘의 집중 시간, 주간 통계, 일정과 투두를 업무자가 바로 읽을 수 있게 요약합니다.

M

회의 기록

음성을 텍스트로 변환하고 회의 내용을 정리해 나중에 검색 가능한 기록으로 남깁니다.

S

AI 검색

회의록과 업무 자료를 기반으로 질문에 답하고 관련 출처를 함께 제공합니다.

G

분석 리포트

집중 이벤트와 작업 패턴을 분석해 개선 포인트를 시각적으로 확인할 수 있습니다.

A

인증 기반 API

JWT와 Google OAuth를 사용하는 프론트/백엔드 분리형 웹 서비스 구조입니다.

사용자 흐름

로그인부터 모니터링, 기록, 검색까지 이어지는 서비스를 짧은 단계로 정리했습니다.

01

로그인

이메일 또는 Google 계정으로 접속하고 개인별 기록을 분리합니다.

02

세션 시작

업무나 학습 세션을 시작하면 웹캠 기반 상태 분석이 함께 동작합니다.

03

기록 수집

집중 이벤트, 회의 텍스트, 첨부 자료를 백엔드에 저장합니다.

04

AI 활용

누적된 기록을 기반으로 업무 맥락 검색과 AI 답변을 제공합니다.

기술 스택

원본 레포의 프론트엔드/백엔드 기술 구성을 소개 페이지에 맞게 압축했습니다.
