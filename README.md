# WorkFocus Landing

Static landing page for the WorkFocus concept site.

## Source

- `index.html`: the full landing page source, including HTML, CSS, and JavaScript.

## Deployment

This repository is ready for static hosting services such as GitHub Pages or Vercel.

### GitHub Pages

1. Open the repository settings.
2. Go to **Pages**.
3. Set the source to `main` and `/root`.
4. Save the settings.

### Vercel

1. Import this GitHub repository in Vercel.
2. Keep the default static site settings.
3. Deploy.
